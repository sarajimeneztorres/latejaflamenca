# 💃 La Teja - Asociación Flamenco Cultural

Sitio web oficial de **La Teja**, una asociación dedicada a la preservación, difusión y disfrute del arte flamenco en Huétor Santillán (Granada). 

Este proyecto es una landing page (*one-page*) diseñada para ser moderna, rápida y completamente responsive. Muestra la información esencial de la asociación: su identidad, las clases que ofrecen, próximos eventos y vías de contacto.

## ✨ Características Principales

* **Diseño "One-Page":** Navegación fluida en una sola página con efecto *Smooth Scrolling*.
* **Responsive Design:** Adaptable a cualquier dispositivo (móviles, tablets y escritorio) mediante Flexbox.
* **Efecto Parallax:** Fondo fotográfico fijo con superposición de capas translúcidas (overlays) para resaltar el contenido y mejorar la legibilidad.
* **Integración de Google Maps:** Mapa interactivo embebido directamente en la sección de contacto.
* **Iconografía Ligera:** Uso de iconos SVG (Instagram) para optimizar el rendimiento sin depender de librerías externas.
* **Rendimiento Óptimo:** Construido únicamente con HTML5 y CSS3 puro, sin frameworks pesados, garantizando tiempos de carga casi instantáneos.
